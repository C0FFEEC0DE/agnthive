agnthive

Claude Code with a software engineering discipline built in. agnthive wraps every prompt in a hook-gated SDLC — discover → design → implement → verify → review → docs — so eight specialist agents collaborate with guardrails on, and a benchmark suite catches regressions before they ship. 

claude plugin marketplace add C0FFEEC0DE/agnthive && claude plugin install agnthive@agnthive

Self-hosted marketplace — add the repo and install in one line. Update anytime with claude plugin update agnthive@agnthive.

View on GitHub Docs

Usage

Address specialist agents with @aliases in any Claude Code prompt:

@e explore the auth module
@cr review api.mjs
@t write tests for utils
@manager implement new feature: user authentication

Specialist agents

@m Manager @e Explorer @a Architect @bug Bugbuster @dbg Debugger @t Tester @cr Code Reviewer @doc Docwriter
Add the marketplace and install directly — restart Claude Code, then see the plugin README for configuration and the optional status line.
