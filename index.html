Claude Code plugin

agnthive.run

A hook-gated SDLC profile for Claude Code — Node hooks enforce discover → design → implement → verify → review → docs, with eight specialist agents and a benchmark suite that catches regressions on every PR.

Install 
claude plugin install ./plugins/agnthive
View on GitHub Docs

What it gives you

  • Deterministic handoff & stop contracts

    Footer markers, specialist-role enforcement, and verification gates are checked by Node hooks — not vibes.

  • Defense-in-depth command blocking

    A command policy blocks dangerous tool calls with exec-form integrity and path resolution, advisory or enforce mode.

  • Token-spend discipline

    Minimal-context agents and explicit role markers keep spend predictable.

  • Benchmark-driven regression coverage

    Per-role smoke tasks and a two-slot CI gate catch agent regressions before they land.

Specialist agents

@m Manager @e Explorer @a Architect @bug Bugbuster @dbg Debugger @t Tester @cr Code Reviewer @doc Docwriter
